faça o clone do seu 
git clone https://github.com/seu_usuario/horario.git

Criar Atualização remota ( apenas uma vez)
git remote add upstream https://github.com/mlealprof/horario.git


Para atualizar
git fetch upstream
git merge upstream/master


Enviar do seu computador para o seu repositório
git add .
git commit -m "Comentário do commit"
git push

- depois entre na sua conta do github e faça um pull request


Volta para o Master
git checkout master 



git push origin master
